Saving plots to a graphics file
-------------------------


```r
pdf("pdf_test.pdf")
plot(1:10, rnorm(10))
dev.off()
```



```r
plot(1:10, rnorm(10))
dev.copy(jpeg, filename="copied.jpg", quality=90)
graphics.off()
```
